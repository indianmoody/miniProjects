This is an impementation of Generic Single Linked List with useful methods. 

Methods:
size(): returns size of List
first(): returns first element/ head of List without removing it
last(): returns last element / tail of List without removing it
isEmpty(): returns true if List is empty otherwise false
addFirst(): appends specified element before the List
addLast(): appends specified element after the List
removeFirst(): removes first element/ head of the List
removeElement(): removes specified element from the List(if present)
