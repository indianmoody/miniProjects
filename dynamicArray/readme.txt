This is an impementation of Dynamic Array List with useful methods. 

Methods:
size(): returns size of List
get(int i): returns specified element of List without removing it
isEmpty(): returns true if List is empty otherwise false
add(int i): appends specified element at the end of List
add(Integer d, int i): add specified element at mentioned position while shifting other elements of List
set(Integer d, int i) : Replace element at index i with d
remove(int i): removes element at index i and shift List
resize(): doubles the List when it get filled
currentLimit() : return current limit/capacity of List
