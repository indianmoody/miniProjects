"ArrayQueue.java": This is am implementation of Queue data structure using an array. 

The implemented methods are:
enqueue(Integer x) : assigns x to the last of Queue and increases the size
dequeue() : returns and deletes data on front of Queue and decreases the size
first() : return data on front of Queue. doesn't delete the data
size() : returns size of Queue
isEmpty() : returns 'true' if Queue is empty otherwise 'false'
