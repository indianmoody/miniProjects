This is an impementation of Array based Deque with useful methods. 

Methods:
size(): returns size of Deque
first(): returns front element of Deque without removing it
last(): returns last element of Deque without removing it
isEmpty(): returns true if Deque is empty otherwise false
addFirst(): appends specified element before the Deque
addLast(): appends specified element after the Deque
removeFirst(): retuns and removes front element of the Deque
removeLast(): returns and removes last element from the Deque
